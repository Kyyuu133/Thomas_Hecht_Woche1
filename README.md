# Rezept Webseite 🍝

Dieses Projekt ist eine einfache Webseite über ein Rezept.  
Die Seite wurde mit **HTML und CSS** erstellt und dient als Übung für grundlegende Webentwicklung.

## Inhalt der Webseite

Die Webseite enthält folgende Bereiche:

- **Navigation** zum Wechseln zwischen den Abschnitten
- **Rezeptbeschreibung**
- **Zutatenliste**
- **Kontaktformular**, über das Besucher Feedback senden können

## Verwendete Technologien

- HTML5
- CSS3

## Funktionen

- Strukturierte Webseite mit mehreren Sections
- Navigation innerhalb der Seite
- Externes CSS Stylesheet
- Layout mit **Flexbox**
- **Hover-Effekt** in der Navigation
- Einfaches **Kontaktformular**
